Initialize the website

Prereq

  • Hugo installed on your machine (see this guide. The installation on Windows is tedious and the only way I could make it work was R > install package blogdown > blogdown::install_hugo() and manually add its path to the system environment variables.)
  • git installed on your machine (see this guide).

Initialize and first deploy

say it is exactly what hugo academic suggests

fork to your github account (put link)

clone your own repo locally

git clone https://github.com/fontaine618/academic-kickstart.git DemoWebsite
cd DemoWebsite

import the academic theme

git submodule update

produce html files

cd ../..
hugo

serve locally to see ythe basic site provided

hugo server
